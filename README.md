# PRE-PAIR PROJECT
  Pre-Pair is a matchmaking platform designed to bring together people who enjoy playing games.

PROGRAMMING LANGUAGE:  

Python

WEB FRAMEWORK: 
 
 Flask

DATABASE:

 Firebase (Real-time, Cloud-based)

FRONTED TECHNOLOGIES:

 HTML,
 CSS

ALGORITHMS:

 User login,
 User registration,
 Password reset
 Game addition,
 User Search,
 User matching (Main project objective)
